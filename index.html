Penjelasan

Penjelasan Program Input Program

File Header yang saya gunakan pada projek kali ini adalah #include (iostream),using namespace std dan int main.Beberapa variable dan tipe data di antaranya adalah variable int,char dan string.Tipe data swicth() dan dimana didalamnya terdapat dua pilihan yaitu case 1 dan case 2 dan break,terdapat juga if else dan yang terakhir adalah return 0 yang terletak pada akhir coding.

Penjelasan Program Output Program

Pada projek program saya buat kali ini adalah berkaitan dengan toko laptop dan khusus untuk satu merek laptop saja yaitu dari MSI.Pada output pertama yang akan muncul di layar adalah pilihan jenis laptop yang akan di beli oleh pelanggan.Terdapat dua jenis laptop yang saya input pada program kali ini,yaitu Gaming Series dan Business Series.Pada setiap jenis laptop tersebut,saya menginput 4 laptop dan tertera harga nya.User akan di beri pertanyaan apakah ingin membaca spesifikasinya atau tidak,apabila user ingin membaca spesifikasinya maka lanjutkan dengan menginput huruf y (yes) dan apabila tidak maka input n (no).Apabila user ingin membaca spesifikasinya maka akan diberi pertanyaan kembali,ingin laptop dengan nomor berapa ?,setelah user menginput angka yang user inginkan maka akan muncul spesifikasi singkatnya di antaranya adalah Prosesor,Sistem Operasi (OS),VGA dan Layar.Setelah membaca spesifikasi tersebut maka user di beri pertanyaan lagi apakah ingin melanjutkan transaksi atau tidak, sama seperti pertanyaan sebelumnya user hanya diminta menginput y / n agar melanjutkan transaksi.Apabila lanjut maka akan ada pertanyaan lagi yaitu ingin membeli laptop dengan laptop nomor berapa dan jumlah laptop yang ingin dibeli setelah itu program akan menotalkan harga dan memuncukan total yang harus dibayar oleh pelanggan.

Ouput Program

Ini adalah tampilan awal ketika software ini di jalankan,dimana pengguna/user akan diberi pilihan antara laptop gaming atau laptop bisnis,dan user harus memasukkan pilihan dengan no 1 atau no 2 lalu enter.

Tampilan ini akan muncul ketika pengguna/user memilih salah satu jenis laptop yang ingin dia cari dan akan muncul list laptop,seri laptop beserta harga laptop tersebut.

Tampilan muncul apabila user/pengguna memilih untuk membaca spesifikasi dari laptop yang user/pengguna pilih dimana akan ditampilkan spesifikasi yang terdiri dari Prosesor,Sistem Operasi,VGA dan Layar Laptopnya.

Ini adalah tampilan final dari software yang telah saya buat setelah menginput semua hal yang berkaitan dengan pilihan laptop di atas,dan sesudah menginput data maka akan muncul total harga dan tampilan masukkan uang bayar yang dimana hal tersebut akan di isi oleh user dengan nominal uang sesuai dengan nominal uang yang dibayarkan oleh pembeli laptop kepada user/pengguna program ini,setelah itu program akan menampilkan jumlah uang kembalian.

Sama halnya dari penjelasan sebelumnya ini juga merupakan tampilan final namun bedanya Ini adalah tampilan final dari software apabila user memilih untuk tidak membaca spesifikasi dari laptop yang telah dipilih.
